Langkah 1: Buat Google Spreadsheet

Buat spreadsheet baru di Google Sheets untuk menyimpan skor semua siswa.

1️⃣ Buka sheets.google.com
2️⃣ Klik tombol + untuk buat spreadsheet baru
3️⃣ Beri nama sheet: "MolecuQuest Leaderboard"
4️⃣ Di baris pertama, isi header: Waktu | Nama | Skor | Mode | Akurasi
5️⃣ Ubah nama tab bawah (Sheet1) menjadi "Leaderboard"
⚙️

Langkah 2: Buat Google Apps Script

Script ini akan menjadi "jembatan" antara game dan spreadsheet kamu.

1️⃣ Di spreadsheet, klik menu Ekstensi → Apps Script
2️⃣ Hapus semua kode yang ada
3️⃣ Copy kode di bawah ini dan paste ke editor
function doPost(e) { var sheet = SpreadsheetApp .getActiveSpreadsheet() .getSheetByName("Leaderboard"); var data = JSON.parse(e.postData.contents); sheet.appendRow([ new Date().toLocaleString("id-ID"), data.nama, data.skor, data.mode, data.akurasi + "%" ]); return ContentService .createTextOutput(JSON.stringify({status:"ok"})) .setMimeType(ContentService.MimeType.JSON); } function doGet(e) { var sheet = SpreadsheetApp .getActiveSpreadsheet() .getSheetByName("Leaderboard"); var rows = sheet.getDataRange().getValues(); var result = rows.slice(1) .filter(r => r[1]) .map(r => ({ waktu: r[0], nama: r[1], skor: r[2], mode: r[3], akurasi: r[4] })); result.sort((a,b) => b.skor - a.skor); var top = result.slice(0, 30); return ContentService .createTextOutput(JSON.stringify(top)) .setMimeType(ContentService.MimeType.JSON); }
🚀

Langkah 3: Deploy sebagai Web App

Deploy script agar bisa diakses dari game.

1️⃣ Klik Deploy → New Deployment
2️⃣ Klik ikon ⚙️ lalu pilih Web App
3️⃣ Execute as: pilih Me
4️⃣ Who has access: pilih Anyone
5️⃣ Klik Deploy → izinkan akses
6️⃣ Copy URL yang muncul (panjang, diawali https://script.google...)
🔗

Langkah 4: Tempel URL ke Game

Paste URL deployment Apps Script di sini.

URL disimpan di browser ini. Bagikan file HTML yang sama ke siswa — mereka tidak perlu setup ulang jika kamu sudah simpan URL di file-nya.
